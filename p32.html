Write a program which performs the following tasks: _ Create a function called input to take an integer array of 10 elements − pass the entire array to a function modify( ) − in modify( ) multiply each element of array by 3 print the new array elements in the modify( ) function DO NOT RETURN BACK TO THE MAIN FUNCTION Use the following function prototypes: void input(); int modify(int a[]);

#include
void input();
void modify(int a[]);
int main()
{
input();
}
void input()
{
int a[10];
printf("Enter 10 numbers: ");
for (int i = 0; i < 10; i++)
{
scanf("%d", &a[i]);
}
modify(a);
}
void modify(int a[])
{
for (int i = 0; i < 10; i++)
{
a[i] *= 3;
printf("%d\n ", a[i]);
}
}
